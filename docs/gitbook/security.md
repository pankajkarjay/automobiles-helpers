# Security and Legal Scope

This project is a demo for authorized control only.

## Not supported

- Unauthorized access to third-party vehicles
- Signal jamming/spoofing/override behavior
- Immobilizer bypass or OEM security bypass

## Recommended production controls

- OEM API integrations and signed firmware channels
- Mutual authentication and rolling challenge/response
- End-to-end encryption and command authorization policy
- Audit logging and revocation controls
